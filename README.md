Nombre del proyecto: Sinfonía Gourmet
Integrantes: Brandon Lozano, Alisson Mediavilla
Temática elegida: Tienda de café y productos gourmet
Descripción del proyecto: Sinfonía Gourmet es un sitio web responsive que simula un restaurante–tienda gourmet. Incluye navegación móvil, menú filtrable por categorías, galería de productos, y un sistema de reservas con resumen en tiempo real, validación de datos y mensajes de confirmación/error. Desplegado en GitHub Pages.
